## README

# This repo corresponds to a paper on male phenology in the North American deer mouse, currently in review at Ecosphere.
# It also contains data from a previous study (McLean BS, Guralnick RP. Digital biodiversity data sets reveal breeding phenology and its drivers in a widespread North American mammal. Ecology. 2021 Mar;102(3):e03258)

# DATA
male_data_ClimNA.csv	final dataset of male reproductive traits and associated climate variables, etc as used in the paper
phen.final.data.csv	final dataset of female reproductive traits and associated climate variables, used in McLean and Guralnick (2020)

# CODE
R code used in the male paper, largely for hierarchical GAMs and generalized linear mixed models
